# Fundamentos del Voleibol

Una aplicación educativa desarrollada en Flutter para explicar de manera clara e interactiva los conceptos básicos del voleibol. 

## Características principales

La aplicación está dividida en tres secciones principales:

*   **Posiciones:** Una guía visual y detallada sobre las seis zonas de la cancha (Zonas 1 a 6), explicando qué función cumple cada posición (Armador, Libero, Puntas, Centrales y Opuesto) y las diferencias entre los jugadores delanteros (ataque) y zagueros (defensa).
*   **Rotación:** Una explicación paso a paso de cómo y cuándo ocurre la rotación en el voleibol. Incluye información sobre el sentido de las agujas del reloj, la obligatoriedad de mantener el orden al momento del saque y las penalizaciones por faltas de rotación.
*   **Saque:** Un resumen de las reglas del saque, quién debe realizarlo, el límite de tiempo (8 segundos) y la posición del jugador detrás de la línea de fondo.

## Cómo ejecutar la aplicación

Para ejecutar este proyecto, asegúrate de tener Flutter instalado y configurado en tu entorno de desarrollo.

1.  Clona el repositorio.
2.  Ejecuta `flutter pub get` para instalar las dependencias.
3.  Ejecuta `flutter run` para iniciar la aplicación en tu dispositivo o emulador.
